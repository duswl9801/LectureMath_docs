Input/Output Path

Name DataType Description
VIDEO_DATABASE_PATH str Relative path of file which has lecture videos' database
VIDEO_FILES_PATH str Relative path of folder where lecture videos are located
Current location: lecturemath/ACCESS2021_release
OUTPUT_PATH str Relative path of directory where results will be located
VIDEO_BASED_ANNOTATIONS int Set to "1"(True) to load the video directly |
Set to "0"(False) to load sequence of images from OUTPUT_FRAME_EXPORT when your video has synchronization issues after you start jumping around the video.
FRAME_EXPORT_FPS float FPS used to extract images from original video
OUTPUT_FRAME_EXPORT str Relative path where VOC images from video will be stored
OUTPUT_FRAME_EXPORT_FORMAT float Format of VOC images
OUTPUT_FRAME_EXPORT_QUALITY float Quality of VOC images
